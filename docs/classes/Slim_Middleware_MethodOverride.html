    HTTP Method Override

    This is middleware for a Slim application that allows traditional desktop browsers to submit psuedo PUT and DELETE requests by relying on a pre-determined request parameter. Without this middleware, desktop browsers are only able to submit GET and POST requests.

    This middleware is included automatically!

    Call

    call() : \array[status,

    Implements Slim middleware interface. This method is invoked and passed an array of environemnt variables. This middleware inspects the environment variables for the HTTP method override parameter; if found, this middleware modifies the environment settings so downstream middleware and/or the Slim application will treat the request with the desired HTTP method.
