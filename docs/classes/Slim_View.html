    Slim View

    The View is responsible for rendering and/or displaying a template. It is recommended that you subclass View and re-implement the View::render method to use a custom templating engine such as Smarty, Twig, Mustache, etc. It is important that View::render return the final template output. Do not echo the output.

    Set data

    setData() : void

    This method is overloaded to accept two different method signatures. You may use this to set a specific key with a specfic value, or you may use this to set all data to a specific array.

    USAGE:

    View::setData('color', 'red'); View::setData(array('color' => 'red', 'number' => 1));

    Exceptions

    \InvalidArgumentException If incorrect method signature
